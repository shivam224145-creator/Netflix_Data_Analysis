# Netflix_Data_Analysis
This project analyzes Netflix dataset to uncover insights on content trends, genres, and country-wise distribution. Using Python, Pandas, and visualization tools, it explores patterns in movies and TV shows, helping understand Netflix’s content strategy and growth over time.

# 📊 Netflix Data Analysis Project

## 📌 Overview
This project focuses on analyzing Netflix dataset to extract meaningful insights about content distribution, trends, and user preferences. The analysis helps understand how Netflix content varies across countries, genres, and time.

## 🎯 Objectives
- Analyze Movies vs TV Shows distribution
- Identify top genres and categories
- Explore country-wise content production
- Understand content trends over the years
- Find most frequent directors and actors

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib & Seaborn
- Jupyter Notebook

## 📂 Dataset
- Source: Netflix Dataset (CSV format)
- Contains details like title, director, cast, country, release year, rating, and duration

## 📊 Key Insights
- Movies dominate over TV Shows on Netflix
- USA contributes highest content
- Drama and Comedy are most popular genres
- Significant growth in content after 2015
